**Monitoramento de Consumo de Eletricidade:**

Este projeto é um sistema web voltado para a gestão eficiente e análise do consumo de energia elétrica, permitindo que os usuários monitorem, estabeleçam metas e identifiquem potenciais desperdícios.

**Funcionalidades:**
O sistema oferece um conjunto completo de ferramentas para o controle de energia:

**1. Monitoramento e Visualização:**

Registro de Consumo: Acompanhamento e registro do consumo de eletricidade em intervalos configuráveis, suportando coleta manual ou integração com medidores.

Dashboards: Permite a visualização de valores de consumo selecionáveis pelo usuário.

Histórico: Armazenamento e visualização do histórico de consumo completo.

Comparativo: Possibilita a comparação do consumo atual com períodos anteriores.

**2. Relatórios e Análise:**

Geração de Relatórios: Criação de relatórios consolidados nos formatos diário, semanal e mensal.

Análise de Desperdício: Identificação de variações atípicas no consumo para sinalizar possíveis desperdícios.

Exportação: Permite a exportação de relatórios e dados históricos em formatos como PDF.

**3. Gestão e Controle:**

Definição de Metas: Configuração de metas mensais de consumo de energia pelo usuário.

Alertas: Emissão de notificações personalizadas quando o consumo ultrapassa os valores das metas estabelecidas.

Personalização: Permite a customização de metas, unidades de medida e alertas.

**Requisitos de Qualidade (Não Funcionais):**

O desenvolvimento priorizou a robustez e a experiência do usuário:

Usabilidade (UX/UI): Interface intuitiva e de fácil uso, em conformidade com boas práticas de design, legibilidade e hierarquia visual.

Desempenho: Resposta a consultas em até 1 segundo (em 95% das requisições).

Compatibilidade: Operação em múltiplas plataformas (desktop e móvel) com interface responsiva.

Disponibilidade: Acesso a funcionalidades básicas, como histórico, mesmo em modo offline.

**COMO USAR O PROGRAMA**

Após baixar os arquivos, abra o index.html para abrir na aba principal do projeto. A partir desta página, é possível mudar pela barra de navegação para transicionar por outras abas, acessando outras possibilidades.


**Tecnologias:**

Frontend: HTML, CSS, JavaScript

Gráficos: Chart.js

Exportação: html2canvas, jspdf

Estilização: Bootstrap
